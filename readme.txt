Aqui voy a explicar como voy montando el proyecto.

Node -v: v22.19.0

Con (mkdir) he creado el directorio portfolioweb y dentro de el en la consola (npm create vite@latest .)

con npm run dev abrirá en proyecto en un servidor en local para ver cambios al momento

Dependencias instaladas:

- (npm install react-router-dom) para utilizar rutas.
- (npm install dotenv) para tener claves guardadas.


Despues he añadido en la carpeta public el diseño en png del portfolioweb
En la Carpeta /src:
- /assets, donde van todas las fotos (/images), fuentes(/fonts) y estilos(/styles)
- /components donde iran los componentes por separado para reutilizarlos
- /pages donde iran todas las paginas de la web
